# StatefulScanner

![CI Pipeline](https://github.com/it-different-courses-com/stateful-scanner/actions/workflows/ci.yml/badge.svg)

Spring Boot 4.0.3 web application using Java 25.

## Build and Run

```bash
# Build and run tests
./mvnw clean install

# Run the application
./mvnw spring-boot:run

# Run tests only
./mvnw test
```
